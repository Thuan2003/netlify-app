Để kiểm tra mình đã add thành công hay chưa các bạn gõ git status lại để check Tiếp theo các bạn gõ lệnh git commit -m "Add new project food" . Lệnh này dùng để note lại các thay đổi để dễ dàng theo dõi nhe. Nếu github yêu cầu xác nhận tài khoản thì bạn thêm thông tin email và tên tk git config --global user.email "you@example.com" git config --glob
